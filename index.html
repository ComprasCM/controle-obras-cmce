Conectar ao Banco de Dados

Insira as credenciais do seu projeto Supabase para salvar e sincronizar os dados online.

1. Acesse supabase.com e crie um projeto gratuito
2. No painel do projeto, vá em Settings → API
3. Copie a Project URL e a anon public key
4. Cole abaixo e clique em Conectar

Settings → API → Project URL
Settings → API → Project API keys → anon public
